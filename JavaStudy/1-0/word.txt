■ OS
OS（Operating System）システム全体を管理し、さまざまなアプリケーションソフトを動かすための最も基本的なソフトウェア
Windows　マイクロソフト社
- 安い
- パソコンのデザインの幅が広い
- 対応しているフリーソフトが多い
- 対応しているゲームが多い
macOS (Mac OS X)　アップル社
- デザインがスタイリッシュ
- デザイン系ソフトと相性が良い
- 画面表示がWindowsと比べて綺麗
- iPhoneなどのApple製品と相性が良い
UNIX　UnixはOSの一種で、現存する中でもっとも古いOSのひとつ。古いだけあって、色々なOSの土台になったOS。
- 安全性が高い。
- 余計な機能が少ない。
- 軽いことがあげられる。
- 安い
- 黒い画面でキーボードだけ操作
- 企業が作成著作権あり
Linux
- 安定的に動く
- 費用がかからない
- 黒い画面でキーボードだけ操作
- 個人が作成。著作権なし
■ 言語・環境
●マークアップ／フロントエンド
HTML　正式名称は「HyperText Markup Language（ハイパーテキスト・マークアップ・ランゲージ）」Webページの土台を作るための言語。見た目を変える。
CSS　正式名称は「Cascading Style Sheets（カスケーディングスタイルシート）」ページのレイアウトを形成するためのスタイルシート。HTMLでマークアップして、CSSでレイアウトを整える。
JavaScript　Webページに動きをつける。
●サーバーサイド
.NET（C#, VB）「Microsoft .NET Framework」のこと。アプリケーション、または開発環境を指す単語。
- C#　.NET Framework上で使うことを想定して設計されている。
- VB　.NET Framework上で使うことを想定して設計されている。プログラミング言語文法がシンプルで、簡単なプログラムを組みたいときに重宝するが、高度な技術を要する分野においては不向きという特徴がある。C#と似た文法を使っているため、VBはWebやスマートフォン向けのアプリケーション、ゲーム開発などに適した言語。
Java
- プログラミング言語のひとつ
- WindowsやMacOS、Linux、UNIXなど様々なマシンで、どこでも動作する
- 様々なものが開発可能
	- TwitterやEVERNOTEなどのWebサービス
	- デスクトップ向けのアプリケーション開発（OpenOffice、MINECRAFT）
	- Androidアプリ
	- 組み込み系（Blue-rayディスクプレーヤー、カーナビ）
	- 物のインターネット(IoT - Internet of Things)」など
Java SE
- 「Javaの基本機能をまとめたもの」
Java EE（Jakarta EE）
- 「JavaSEを元にしてサーバーサイドの機能を追加したもの」要するにJavaEEは簡単にいえば「JavaSE+拡張機能」
Perl
- 特徴・高い後方互換生・テキスト処理に優れている・可読性が高い・コンパイルが不要・CPANモジュールが豊富
- どんな開発？・サーバー管理|・Web開発
PHP
- 	特徴：Webアプリ・サービス開発に特化した言語
- 	メリット：構文がシンプルで簡単、PHPは人気言語ためWEB上に情報が多い、比較的習得難易度が低い、需要が高いので、案件が豊富
- 	デメリット：Webアプリ・サービス以外のものはほぼ作れない、セキュリティが甘め、自由度が高すぎる
Ruby
- 	特徴：Webアプリケーションを作成するためのフレームワーク（枠組み）
- 	得意：ショッピングサイト、ブログサイト、アプリケーションサイトなど、フレームワークを使い構築できるWebサイトです。
- 	メリット 自由度が高い Ruby on Railsとの相性が素晴らしい 記述がしやすい
- 	デメリット： 実行速度が遅い 自由度の高さが仇となることも
- 	代表的なフレームワーク： Ruby on Rails　Hanami Sinatora Padrino

SQL
	- DB2：「IBM Db2 Family」とも呼ばれる
	  	アーキテクチャ、オプティマイザ（一般的にコスト・ベース）、読み取り一貫性、IBM Db2はOracle Databaseの約2分の1程度の金額、独自に新しい技術を発明する
	- MySQL：GPLと商用のデュアルライセンス（実質的に、MySQLはOracleによって開発・ソースコードを管理されている）、企業は商用ライセンスがあるため安心して導入できる
		- Maria DB：MariaDBはMySQLから派生したRDBMS（リレーショナルデータベース管理システム）、完全なるGPLライセンス、「MariaDBの方が使いやすい」、「カラム型エンジン（大規模なデータやビッグデータなどの集計を得意）」、MySQLと非常に高い互換性を持っている
	- Oracle DB：M&Aで既存の技術を融合し新しいサービスを生み出す
